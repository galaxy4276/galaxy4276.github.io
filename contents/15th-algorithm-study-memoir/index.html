[백준] 쇠막대기

10799. 쇠막대기

() 를 레이저로 취급하여 총 몇 개의 파이프가 절단되었는 지 갯수를 세는 스택 문제이다.

#include <iostream> #include <bits/stdc++.h> #include <stack> using namespace std; int main() { ios::sync_with_stdio(false); cin.tie(nullptr); stack<char> S; string input; int total = 0; cin >> input; for (int i = 0; i < input.length(); i++) { if (input[i] == '(') { S.push(input[i]); } else if (input[i] == ')' && input[i - 1] == '(') { S.pop(); total += S.size(); } else { S.pop(); total++; } } cout << total << endl; }

입력 값 string 을 순회 돌려서

'(' 일 시 stack 에 넣고

if (input[i] == '(') { S.push(input[i]); }

'()' 인 경우를 판단해 stack 의 사이즈 만큼 더하고

else if (input[i] == ')' && input[i - 1] == '(') { S.pop(); total += S.size(); }

')' 일 경우 파이프를 자르면 한 개가 더 남는 원리로 +1 을 수행한다.

else { S.pop(); total++; }

[LeetCode] Reverse Linked List

206. Reverse Linked List

이 문제는 저번에도 풀었던 건데 늘 어떻게 풀었는 지 원리를 생각하면 잘 생각이 안난다.. 참조를 왔다갔다 하는 구조가 늘 머리에 안익는다.

#include <iostream> #include <bits/stdc++.h> #include <stack> using namespace std; struct ListNode { int val; ListNode *next; ListNode() : val(0), next(nullptr) {} ListNode(int x) : val(x), next(nullptr) {} ListNode(int x, ListNode *next) : val(x), next(next) {} }; class Solution { public: ListNode* reverseList(ListNode* head) { ListNode *nextNode, *prevNode = nullptr; while (head) { nextNode = head->next; head->next = prevNode; prevNode = head; head = nextNode; } return prevNode; } }; }
